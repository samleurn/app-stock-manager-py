## Requirements

```
# Enviroment Ambient
python venv .venv
.venv/bin/active
```

```
# Instalation Guide
pip install "uvicorn[standard]" "reactpy[fastapi]" "fastapi[standard]"
```

```
# To init the client
uvicorn client.spa:app --reload --port 5050
```

```
# To init the api
uvicorn api.server:app --reload --port 3333
```
